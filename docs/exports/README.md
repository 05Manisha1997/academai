## How to use

### SVG (PowerPoint, Google Slides, Word)
1. Download the `.svg` file
2. Insert → Image → upload SVG
3. Or open in browser → Print → Save as PDF

### draw.io
1. Open [https://app.diagrams.net](https://app.diagrams.net)
2. File → Open → select `academai-technical-architecture.drawio`
3. Edit layers, export as PNG/PDF/SVG

### Markdown pitch
- Open in any editor or paste into Notion, Confluence, or GitHub
- Export to PDF from VS Code / Cursor with a Markdown PDF extension
